## 网页版仿微信聊天功能，包括表情、图片、文字发送 mockjs模拟实时数据

In the project directory, you can run:
### `yarn install`
安装依赖
### `yarn start`
启动项目

### `yarn build`
打包为生产包
### `yarn eject`
